About


Great Battle is a project whose task is to show both team and individual knowledge and capabilities of
each of its participants. Our goal was not just to complete the task assigned to us, but to implement a
full-fledged web application using relevant frameworks and a modern solution to certain problems.
Our application used: the Express framework, the JWT standard, bcrypt functions, mysql database,
websocket and so on.

The backend of the project is written using the node js platform, and the frontend is written using js and a standard markup
language and styles.

Disclaimer: This project was implemented for educational purposes and does not pretend to be commercialized.
